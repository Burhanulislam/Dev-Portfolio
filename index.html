Vanilla JavaScript

Vanilla JavaScript is plain JavaScript without any libraries or frameworks. It’s used to create dynamic, interactive websites directly in the browser, offering full control over web development.

React

React is a JavaScript library developed by Facebook for building fast, dynamic user interfaces. It uses components and a virtual DOM to efficiently update and render web applications.

Node.js

Node.js is a JavaScript runtime built on Chrome's V8 engine, used to build fast, scalable server-side applications, especially APIs and real-time services like chat applications and streaming.

MongoDB

MongoDB is a NoSQL database that stores data in flexible, JSON-like documents. It’s scalable, schema-less, and ideal for modern applications needing fast, efficient handling of large, unstructured data.
